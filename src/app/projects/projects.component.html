Gothello

Gothello is an application I wrote to get familiar with the popular Go programming language. It is an implementation of the popular board game Othello using Go and basic html/js. Rules and game logic are enforced serverside. The game uses a REST API where possible and websockets otherwise. It is hosted on heroku and can be accessed here. Do note that it is incomplete at this time, but basic functionality such as being able to play against another player is available.
